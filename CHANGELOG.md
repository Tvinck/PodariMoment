# Changelog

Все значимые изменения проекта ПодариМомент.
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/).

## [1.0.0] — 2026-05-29 (Первый релиз)
### Добавлено
- Голос для гендер-пати: 3 тарифа (Базовый 399 / Премиум 599 / VIP 999 ₽)
- Оплата через Т-Банк эквайринг (идемпотентность, проверка подписи)
- Автогенерация голоса через Kie.ai (ElevenLabs), webhook + cron-добор
- Личный кабинет клиента (поиск по email)
- Публичная страница трекинга `/track`
- Панель администратора: заказы, CRM-воронка, финансы, промпты, голоса, цены, Kie.ai
- Email-уведомления через Resend (принят / готов / ошибка)
- Промокоды и скидки (percent / fixed)
- Страницы `/faq` и `/about`, страница 404
- SEO (OG, Twitter, canonical, manifest, favicon), Яндекс.Метрика
- Telegram-кнопка поддержки

## [0.9.0] — 2026-05-24 (Beta)
### Добавлено
- SEO-мета, аналитика, валидация формы заказа
- Редактор промптов и голосов в админке
- Финансовая аналитика (график выручки, KPI, разбивка по тарифам)
- Кэширование данных, error boundary, toast-уведомления

## [0.5.0] — 2026-05-23 (Alpha)
### Добавлено
- Дизайн-система, лендинг с фокусом на гендер-пати
- Форма заказа с voice-preview и Reveal Stage
- Supabase (таблица orders), serverless-функции
- Мультистраничный SPA-роутинг (History API)
